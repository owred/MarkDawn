# Шпаргалка по Markdown

## Заголовки

``` sh
# H1
## H2
### H3
#### H4
##### H5
###### H6   
 ```

> Кроме того, H1 и H2 можно обозначить подчеркиванием:
``` sh
Alt-H1
======

Alt-H2
------

H1
H2
H3
H4
H5
H6  
``` 
> Кроме того, заголовки H1 и H2 можно обозначить подчеркиванием:
``` sh
Alt-H1
Alt-H2  
```
